# Introduction to Claude

Claude is a family of large language models developed by Anthropic. Claude models are trained using Constitutional AI, a technique developed by Anthropic researchers to make AI systems safer and more aligned with human values.

The Claude family currently includes Opus, Sonnet, and Haiku models, each optimized for different use cases. Opus is the most capable, designed for complex reasoning tasks. Sonnet balances capability and speed. Haiku is the fastest and most cost-effective, ideal for high-volume applications.

Claude models support a context window of up to 200,000 tokens, allowing them to process roughly 150,000 words of text in a single request. This makes Claude well-suited for analyzing long documents, codebases, and conversations.

Claude can be accessed through the Anthropic API, the claude.ai web interface, and various third-party platforms. The API supports streaming responses, function calling, vision input, and structured output formats like JSON.

Claude is designed to be helpful, harmless, and honest. It will decline requests that could cause harm, and it acknowledges uncertainty rather than fabricating information.
